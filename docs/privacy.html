🌴 PALM Privacy Policy

Last updated: May 1, 2025

PALM (Personal AI Link Manager) is a Chrome extension that helps you save and organize web links with AI-generated summaries. Your privacy is our top priority.

Data We Collect

PALM does not collect, transmit, or store any personal data on external servers. All data stays on your device.

What Is Stored Locally

  • Links you save (URL, title, summary, tags) — stored in chrome.storage.local on your device only
  • Your Gemini API key — stored in chrome.storage.sync (synced across your Chrome profile via your Google account, never sent to PALM servers)
  • Daily usage counter — stored locally to track your API usage

Third-Party Services

When you save a link, the page content and your Gemini API key are sent directly from your browser to the Google Gemini API to generate a summary. This request goes directly to Google — PALM has no backend server and never sees your data.

Google's use of data submitted to the Gemini API is governed by Google's Privacy Policy.

No Backend, No Accounts

PALM has no servers, no accounts, no analytics, and no telemetry. The extension source code is fully open source and available at github.com/abhi-singh-123/palm.

Data Deletion

You can delete all saved links at any time from the PALM dashboard. Uninstalling the extension removes all locally stored data.

Contact

If you have questions about this privacy policy, please open an issue on GitHub.
